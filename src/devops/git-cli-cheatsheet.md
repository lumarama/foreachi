# Git CLI Cheatsheet

Clone repository:

```
git clone <repo-url>
```

Create a new branch to track changes in the remote branch and switch to it. This is what you would normally do to start working with a branch:

```
git checkout -b <local-branch-name> origin/<remote-branch-name>
```
